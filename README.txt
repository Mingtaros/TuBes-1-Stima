Aplikasi Permainan Kartu 24 dengan Algoritma Greedy

Files:
1. Source Codes
    - frontend.py
    - backend.py
    - kartu.py

2. Documentations
    - LAPORAN TUGAS BESAR I IF2211.pdf
    - README.txt

Bahasa Pemrograman : Python 3

Compiling dan Running :
- Pengguna memiliki Python 3 terinstall di komputernya
- Pengguna memiliki Python Library Kivy dan pydealer
- lakukan perintah 'python frontend.py' untuk menjalankan front end 1
- lakukan perintah 'python frontend.py AA BB' untuk menjalankan
  front end 2 (AA dan BB merupakan file eksternal)

Back End :
Aplikasi 24 game solver ini menggunakan algoritma Greedy.


Front End :
1. Front End 1
   Front End 1 berupa GUI yang menampilkan 4 buah kartu dan mencetak hasil
   perhitungan dari back end ke layar.
   - Take Cards = mengambil 4 kartu dari deck
   - Submit     = melakukan perhitungan dari back end
   - Cancel     = mengosongkan kartu di tangan
   - Quit       = keluar dari aplikasi

2. Front End 2
   Front End 2 meminta input pengguna dari file eksternal pertama dan
   mengeluarkan hasil perhitungan dari back end ke file eksternal kedua.


Informasi-informasi tambahan :
1. Kompilasi dan run pada program back end menghasilkan program yang menerima
   4 masukan pengguna dan memprosesnya sesuai dengan algoritma greedy.
2. Program untuk front end 1 dan front end 2 adalah program yang sama
3. GUI yang digunakan diimplementasikan menggunakan library Kivy
